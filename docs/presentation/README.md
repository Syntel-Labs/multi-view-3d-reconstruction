# Presentacion final

Slides y video para la presentacion del 21 de mayo de 2026.

## Estructura

```text
docs/presentation/
├── README.md
├── slides/             # PDF y fuentes (PowerPoint, Keynote, etc.)
└── video/              # video final mp4 + screencasts intermedios
```

## Reglas

- Maximo 10 slides en el PDF final (rubrica del curso).
- Video con duracion acorde a lo pedido por el catedratico.
- Slides PDF se exporta como `slides/final.pdf`. Las fuentes editables van junto a el con el mismo basename (`final.pptx`, `final.key`).
- Screencasts intermedios de la demo en `video/` con nombre `YYYYMMDD_<descripcion>.mp4`.
