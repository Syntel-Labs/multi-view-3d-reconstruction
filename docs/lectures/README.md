# Apuntes de clase

Espacio compartido para los apuntes propios del equipo a lo largo del curso (sintesis, ejemplos resueltos, dudas resueltas en grupo). Es el "pegamento" entre lo que se ve en clase y lo que se construye en el pipeline.

## Diferencia con `docs/theory/`

| Carpeta | Contenido |
| :--- | :--- |
| `docs/theory/` | Copias literales de las anotaciones del catedratico (semana 1 a 7). |
| `docs/lectures/` | Apuntes propios y resumenes elaborados por el equipo. |

Para cruzar las semanas con los modulos del pipeline ver [`docs/course-references.md`](../course-references.md).
